About Error Level Analysis

Understanding the science behind image forgery detection

What is Error Level Analysis?

Error Level Analysis (ELA) is a forensic technique used to identify portions of an image that have been modified or tampered with.

The technique works by exploiting the fact that different parts of a manipulated image often have different compression histories. When an image is saved as a JPEG, it undergoes lossy compression. If parts of the image have been edited or copied from other sources, they will have different error levels when recompressed.

How Does ELA Work?

Step-by-Step Process:
  1. Load Original Image: The suspect image is loaded and converted to RGB format
  2. JPEG Recompression: The image is saved as JPEG with exactly 90% quality
  3. Reload Compressed: The recompressed image is loaded back into memory
  4. Calculate Difference: Pixel-by-pixel differences are computed between original and recompressed versions
  5. Amplify Brightness: The differences are enhanced and normalized for visibility
Key Principles:
  • Compression Artifacts: Different compression histories create detectable patterns
  • Error Levels: Tampered regions show higher error levels than authentic areas
  • Visual Enhancement: Brightness amplification makes subtle differences visible
  • Forensic Evidence: Provides visual evidence of potential manipulation

PS-7 Compliance

This implementation strictly follows the Problem Statement 7 (PS-7) requirements for Error Level Analysis:

Technical Specifications:
  • Fixed JPEG quality at 90% (mandatory)
  • Absolute difference calculation using ImageChops
  • Brightness amplification with proper normalization
  • RGB color space processing
Quality Assurance:
  • Consistent algorithm implementation
  • Proper temporary file handling
  • Error handling and validation
  • Standardized output format

Interpreting ELA Results

Bright/Hot Areas

High error levels indicate potential tampering or editing

Dark/Cool Areas

Low error levels suggest authentic, unmodified content

Uniform Patterns

Consistent error levels across the image indicate authenticity

Limitations & Considerations

Important Disclaimers:
  • Not 100% Accurate: ELA is a forensic tool, not a definitive proof of tampering
  • False Positives: Natural image features can sometimes appear as bright areas
  • Skilled Manipulation: Advanced editing techniques may not be detected
  • Context Required: Results should be interpreted by forensic experts
  • Complementary Tool: Best used alongside other forensic analysis methods
When ELA May Not Work:
  • Images that have been heavily processed or filtered
  • Very low quality or highly compressed source images
  • Images with uniform textures or solid colors
  • Professional manipulations designed to avoid detection
